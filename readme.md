# Bomberman.js

Bomberman.js est une version du célébre jeu bomberman réécrit en javascript (vanillajs).
les ressources (images et sons) proviennent du projet [Bombermaaan](http://bombermaaan.sourceforge.net/).

### Démarrer le jeu


```
npm install       
npm start
npm run server
```
